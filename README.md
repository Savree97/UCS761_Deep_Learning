## Overview

This repository documents a structured academic progression through core deep learning concepts, beginning with single-layer models and extending to deeper neural architectures.

The emphasis throughout has been on understanding:

- How parameters influence predictions  
- How gradients propagate across layers  
- Why depth impacts training stability  
- How optimization affects convergence  

All primary models are implemented manually without high-level deep learning frameworks.
